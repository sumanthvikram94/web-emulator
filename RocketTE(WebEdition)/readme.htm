Welcome to Rocket Terminal Emulator Web Edition 10.2

Introduction

Rocket TE Web is a terminal emulator designed to communicate with today's most popular host systems. It's written entirely in JavaScript and HTML5 and works with modern web browsers and can run on PC’s, tablets or just about any mobile device regardless of the client OS. No software is installed on the client device.

The Rocket TE Web server, connects the web-based client terminal emulator to the terminal host. The server can be installed on any platform that supports Node.js.

Neither the Rocket TE Web client, nor the server, are dependent on any particular client or server operating system.

Supported Emulations

  • IBM System Z - Mainframe (3270)
  • IBM System i - (5250)
  • Linux / VT220 / VT320 / VT420

Installation

Rocket TE Web installation is quick and easy, all you have to do is choose your installation platform and run the corresponding installer.

To get up and running as quickly as possible, we recommend installing Rocket TE Web on a Windows platform.
