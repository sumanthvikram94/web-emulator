Class: RocketTeWebApi

RocketTeWebApi(serverUrl)

The wrapper class of Rocket TE Web API. Referring to naming conventions of Google: https://google.github.io/styleguide/jsguide.html#naming-method-names

Constructor

new RocketTeWebApi(serverUrl)

Parameters:
Name Type Description
serverUrl string URL of the Rocket TE Web server. E.g. 'http://localhost:8543'

(async) playScript(scriptSource) → {object}

Plays the given scriptSource in terminal. Use this function in case your action is not supported by any formal API functions
Parameters:
Name Type Description
scriptSource string String containing script source code. For details about script, refer to the script document: http://localhost:8543/ZLUX/plugins/com.rs.bzw/web/lib/js/scriptsDoc/index.html
Throws:
Will throw error in case Rocket TE Web is not launched
Returns:
Will return an outputs object in case automationObj.outputs is assigned a value inside of the script source
Type
object
Example
const scriptStr = "automationObject.connect(); \n myObj = automationObject.getRowColumn(); \n automationObject.outputs.myRetVal = myObj.row;";
const row = await rtewApi.playScript(scriptStr);
