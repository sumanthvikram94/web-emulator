(generator, inner) sendFunctionYield(functionName, kbRestoreTimesopt, kbRestoreWaitMsopt) → {ScriptWaitState}

For TN3270 and TN5250, executes a function on the terminal and pauses to wait for the terminal to be ready for more input. For VT, executes a function without pause.
Parameters:
Name Type Attributes Default Description
functionName string Function to be performed.
kbRestoreTimes number <optional>
 Times of keyboard restore before executing next function.
kbRestoreWaitMs number <optional>
 2000 Milliseconds to wait for each keyboard restore time specified by kbRestoreTimes parameter. The default value is 2000 (2 seconds). If the keyboard restore doesn't happen after a time period of kbRestoreTimeout milliseconds, it will bypass the current waiting.
Returns:
True if successful, otherwise false.
Type
boolean
Yields:
A wait state of type "messageReadWait" which should be satisfied upon the terminal being ready for input again.
Type
ScriptWaitState
